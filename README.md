# JS Full Stack

## Ferramentas
- Node
```cmd
npm init -y
```

- Express
```cmd
npm install express
```

## Ajustes

- Mudar no `package.json`
```js
  "type": "module",
```

- Para nao precisar ficar reiniciando o servidor
- rodar com `npm run dev`
```json
  "scripts": {
    "dev": "node --watch server.js"
  },
```
